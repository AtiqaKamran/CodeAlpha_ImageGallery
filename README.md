Pixel Gallery is a modern and fully responsive image gallery website built with HTML, CSS, and JavaScript. It features a stunning cover page with a dynamic background slider, category-based filtering, search functionality, lightbox view, dark/light theme toggle, grid/list view options, and an About section.

Key Features
- Full-screen hero section with a dynamic background image slider
- "View Gallery" CTA button and theme toggle
- 40+ Images across 9 categories (Nature, City, Animals, Food, Abstract, Books, Makeup, Clothes)
- Search functionality — search by title or category
- Dynamic Cover Page with background slider, like widget, and smooth animations
- Category Filtering & Search functionality
- Lightbox with navigation, download, and keyboard shortcuts (Escape, Arrow keys)
- Download button to save images
- Dark/Light Theme with persistent localStorage
- Grid/List View toggle
- Responsive Design — works on desktop, tablet, and mobile
- About Section with gallery information and "Made by Atiqa Kamran" credit
- Toast notifications & Back to Top button
